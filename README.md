# Urdu → English Translator

A Python-based **Urdu to English Translator** with a simple **Tkinter GUI**, leveraging **HuggingFace Transformers** and **PyTorch**.  
This project provides a user-friendly interface to translate Urdu sentences into English dynamically, while preserving proper names and technical terms accurately.

---

## Features

- Translate **any Urdu text** into English dynamically.
- Accurate translation of **proper names**, **technical terms**, and common phrases.
- Easy-to-use **Tkinter GUI** with input and output boxes.
- Copy translated text to clipboard or clear text easily.
- Built using **pre-trained AI models**, so no need to train from scratch.

---

## Technologies Used

- **Python 3.x** – Programming language  
- **HuggingFace Transformers** – Pre-trained NLP models  
- **PyTorch** – Backend for running models efficiently  
- **SentencePiece** – Tokenizer for language processing  
- **Tkinter** – GUI library for desktop application
